User Guide
==========

**F1DB Application**

The F1DB application keeps the data of the pilots, cars, tracks and brands. There are 6 main
navigation buttons and 2 buttons for signing in and logging in. In navigatino bar, there is also a search box.
The search can be made for different categories such as cars, engines, pilots, countries, tracks, brands, founders and grands prix.

   :alt: navigation bar
   :align: center

   The navigation bar consist of 8 buttons total and a search bar.
